# 1.简介

这是自己实现的一个单体简单博客平台

主要实现技术：

+ SpringBoot
+ Thymeleaf
+ MyBatis—Plus

+ JDK 1.8

> SQL脚本文件在src\main\resources\SQLData目录下面存放，导入数据库直接执行就好

注意：本项目中有些资源是导入的CDN，有些资源可能以后会挂；有时候访问特别慢，使用科学上网会快些
